1ハッカソン調査・アイデア検討

ハッカソン概要

Microsoft Agent Hackathon powered by Tokyo Electron Device に参加検討。

必須要件：Azure実行基盤（App Service / Container Apps等）＋ Microsoft AI技術（Azure OpenAI等）1つ以上の使用。

作成するものの方向性

① 会社のTeamsボット（会議内容を学習するチャットボット）の理解を深める

② オリジナル：音声で近くの雰囲気ある酒場を案内するエージェント（「創業○年の渋い酒場を探して」などの会話で動作）

2Microsoft 365テナントの準備

問題：Microsoft 365 Developer Program に登録できない

エラーコード 41101：既存テナント（pm111）が期限切れのため登録不可。個人アカウントでも対象外。

解決：新規テナントで無料試用版を取得 ✅

新しいMicrosoftアカウント（onmicrosoft.com）を作成。

Microsoft 365 Business Standard 無料試用版（90日間）を取得 → Teamsが使える状態に。

試用期限：2026年6月19日

Teamsへのアクセス確認

teams.cloud.microsoft からTeamsに正常アクセス。「個人事業」テナントでチャット・チャネルが利用可能な状態を確認。

3Azureサブスクリプションの準備

問題：Azure無料アカウントが対象外

新規テナント・個人アカウントともに「You're not eligible for an Azure free account」と表示。過去に無料枠を使用済みのため。

解決：従量課金（Pay-as-you-go）で登録 ✅

ハッカソン程度の開発であれば実費は数百〜1,000円程度。クレジットカード登録で即利用開始。

サブスクリプション名：Azure subscription 1

Foundryリソース作成 ✅

Azureポータル → Microsoft Foundry → Azure OpenAI → 「Foundry（おすすめ）」を選択して作成。

リソースグループ: Hackathon2026
リソース名:       Hackathon202604
リージョン:       Japan East
デフォルトプロジェクト: proj-default

GPT-4oモデルのデプロイ ✅

Foundryポータル → モデルカタログ → gpt-4o を検索 → 「このモデルを使用する」でデプロイ。

デプロイ名:       gpt-4o
展開の種類:       Standard
プロビジョニング: 成功
場所:             Japan East

APIキーとエンドポイントの取得 ✅

リソース管理 → キーとエンドポイント → OpenAIタブで確認。

エンドポイント: https://[リソース名].openai.azure.com/
デプロイ名:     gpt-4o

キー（SECRET）は .env.local.user に設定済み（GitHubには含めない）。

5VS Code開発環境のセットアップ

Microsoft 365 Agents Toolkitのインストール ✅

VS Code拡張機能マーケットプレイスで「Microsoft 365 Agents Toolkit」をインストール（旧名：Teams Toolkit）。

プロジェクトの生成 ✅

Agents Toolkit → 「新しいエージェント/アプリの作成」→ 以下を選択：

テンプレート: Teams エージェントとアプリ
種類:         一般的なTeamsエージェント
LLM:          Azure OpenAI
言語:         Python

生成先：AgentsToolkitProjects\Hackathon2026（後でプロジェクトフォルダに移動）

環境変数の設定 ✅

ファイル：env/.env.local.user

SECRET_AZURE_OPENAI_API_KEY=（設定済み）
AZURE_OPENAI_ENDPOINT='https://[リソース名].openai.azure.com/'
AZURE_OPENAI_DEPLOYMENT_NAME='gpt-4o'

依存ライブラリのインストール ✅

python -m pip install -r src/requirements.txt

エラーなしで完了。Pythonバージョン：3.13.12
